How does javascript work ?

javascipt is a client-side scripting and interpreted programming language.Most programming languages for the web run on the server.javascript actually run by your browser and that makes it quite uniqe in the world of programming language.JavaScript source code first goes to the browser.Browser codes are given to chrome V8 engine.The engine compile that code. There are two types of compilers. first interpreted compiler and second compiled compiler.Interpreted compiler easy to debug but slow process and compiled compiler fast process but system crash problem or hard to debug.Because of this problem More modern browsers use a technology known as Just-In-Time (JIT) compilation.This JIT compiler more than fast enough.This JIT compiler combination between interpreted and compiled compiler, This compiles JavaScript to executable bytecode just as it is about to run.Then the V8 engine outputs to the computer.At last, user show the code output.
